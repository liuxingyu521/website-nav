# Website nav

用来记录日常工作中用到的网站，方便随时随地查阅。

**网站持续收录ing...**

## 项目开发

本项目基于 vue3 + vite2 搭建，开发流程如下

```shell
# 1. 克隆项目代码
$ git clone git@github.com:liuxingyu521/website-nav.git

# 2. 安装依赖
$ pnpm install

# 3. 启动本地服务
$ pnpm dev
```
